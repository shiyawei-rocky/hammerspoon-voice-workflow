# 文本处理热键速查表

## 基础修正
- **F7** → 校对与轻度润色 (`f7_polish`)
  - 意图：修正错别字、标点、轻微语病，删除明显口语填充
  - 适用：文本已基本成型，做最后校对

- **Alt+W** → 口语转书面优化 (`alt_w_articulate`)
  - 意图：把零散、口语化表达整理成清晰自然的书面文本
  - 适用：整理想法要点、优化长消息/邮件表达

## 专项处理
- **F5** → 语音/听写整理 (`f5_asr_post`)
  - 意图：整理语音或听写文本，纠同音字、加标点、规范格式
  - 说明：系统听写模式为听写输入；录音模式为云端 ASR
  - 可配合【术语表】保证专业词准确

- **F6** → 按指令修改 (`f6_selection`)
  - 意图：按【用户指令】精确修改某处，同时修正语法
  - 格式：`【用户指令】修改要求` + `【原文】待处理文本`

- **Alt+E** → 翻译 (`alt_e_translate`)
  - 意图：自动识别语种并翻译，保留专有名词和代码

## 信息处理
- **Alt+A** → 摘要（剪贴板） (`alt_a_summary`)
  - 意图：生成不超过 4 句话的客观摘要

- **Alt+C** → 仅检查问题 (`alt_c_check`)
  - 意图：列出文本中的错别字、语法、标点等问题，不改写

- **Alt+X** → 批判性分析（剪贴板） (`alt_x_critique`)
  - 意图：深度分析逻辑矛盾、隐含前提，找出根本问题

## 任务规划
- **Alt+Q** → 需求转步骤（剪贴板） (`alt_q_struct`)
  - 意图：将模糊笔记转为可执行的 3–5 步指令
  - 信息不足时会主动指出缺失信息

- **Alt+Z** → 任务拆解 (`alt_z_steps`)
  - 意图：将明确任务描述拆解为 3–7 个可执行步骤，带检查点

## 一键助手
- **F8** → 一键助手
  - `digest` 模式：汇总最近记录并生成简洁摘要
  - `workflow` 模式：跑项目情报流水线，生成总结、分析和归档文件

## 复杂任务处理流程（建议）
1. 描述模糊/信息不全 → 先用 **Alt+Q** 明确需求  
2. 描述清晰完整 → 直接用 **Alt+Z** 拆解步骤  
3. 对方案有疑虑 → 可选 **Alt+X** 深度分析

## 写作质量检查链（可选）
**F7** → **Alt+W** → **Alt+C**
